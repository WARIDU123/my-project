## Project Overview
This project involves designing and developing a fully functional, responsive e-commerce storefront utilizing the **Shopify** low-code/no-code platform. The store, *EcoThread Apparel*, focuses on sustainable fashion, offering user-friendly navigation, seamless cart interaction, and an intuitive user interface (UI) designed to maximize user experience (UX). All project configurations, theme styling, and product management were executed via Shopify’s administrative dashboard.

---

## Features Implemented

### 1. Storefront Pages
* **Homepage:** Features a clean, minimalist design with a prominent header, an engaging welcome announcement banner, the store brand name, and a "Shop Now" call-to-action (CTA) button leading directly to the catalog.
* **Product Page:** Displays a structured grid layout featuring five unique sustainable fashion products. Each item contains high-quality images, comprehensive descriptions, sizing options, and dynamic pricing.
* **About Page:** Shares the brand story, core values, and our mission toward achieving zero-waste fashion production.
* **Contact Page:** Features a functional contact form, store email address, customer service phone number, and social media integration blocks.

### 2. Functional Capabilities
* **Cart Interaction:** Fully operational dynamic cart system. Users can select variants (size/color), click "Add to Cart," see real-time updates via a slide-out drawer cart, change quantities, and proceed to a simulated checkout page.
* **Navigation & UI/UX:** Implemented a sticky header navigation menu allowing seamless switching between pages. Structured the footer to house terms of service, privacy policy, and accepted payment iconography.
